# React Font Previewer

**개인 공부 용도로 작업한 파일 입니다.**
[참고사이트: 스포카 한 산스 기술 블로그] (https://spoqa.github.io/spoqa-han-sans/ko-KR/)

## 적용 기술

**React, Redux, Webpack, Babel** 

## 사용 가능 스크립트

```
npm run dev
```
(http://localohost:3000) 개발 모드

```
npm run build
```
배포용 빌드 명령어
